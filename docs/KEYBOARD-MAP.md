# Keyboard & pointer map — Starfall Salvage

Store accessibility evidence (C15). Touch uses canvas drag + FIRE + flight menu (☰).

| Input | Action |
|-------|--------|
| `W` / `↑` | Move up (desktop) |
| `S` / `↓` | Move down (desktop) |
| `A` / `←` | Lane left |
| `D` / `→` | Lane right |
| `Space` | Phase dash |
| `F` | Fire |
| `P` | Pause / resume |
| `R` | Reset run |
| `Enter` | Start / fly again (when not playing) |
| `[` / `]` | Queue junction turn at corner |
| `Escape` | Close flight menu, ecosystem flyout, leaderboard, account modal |
| Mouse hold (canvas) | Fire while playing |
| Tap canvas | Start / dash (touch) |
| `mobileFireButton` | Fire (touch) |
| `flightMenuToggle` (☰) | Multitask menu: resume, step out, weapon mode |
