# Starfall Source Assets

This folder is for editable art sources and interchange exports.

Keep Blender originals here:

- `*.blend` for student/artist source work
- `*.obj` for the raw WebGL conversion bridge
- `*.glb` only as a source archive until the game has a runtime glTF loader

Runtime files belong in `assets/runtime/`, not here.

## Current Lesson Target

`starfall_boss_drone_blockout.obj` is the teacher reference blockout for Lesson 001. Use it as the scale, orientation, and naming target when building the real Blender boss drone.
